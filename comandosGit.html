Describe que hacen los siguientes comandos de git, escribe su descripcion al frente en una sola linea. git status: Este comando se usa para obtener el estado actual del repositorio git clone: Este comando se usa para clonar repositorios existentes en git hub. git pull: Este comando se usa para obtener el proyecto con las ultimas modificaciones del repositorio principal. git checkout: Este comando se usa para pasar a una rama secundaria o volver a la rama principal. git log: Este comando se usa para ver quien ha realizado cambis en el proyecto. git branch:Este comando se usa para crear una nueva rama. git add:este comando se usas para agregar los cambios realizados al proyecto. git commit: Este comando se usa para guardar el proyecto con los cambios realizados en el area local. git push:Este comando se usa para subir el proyecto con los cambios realizados al repositorio remoto. git merge: Este comando se usa para insertar la rama secundaria con los cambios a la rama master o a otra rama.
